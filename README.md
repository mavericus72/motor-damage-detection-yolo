# 🚗 Car Damage Detection using YOLOv8

## 📌 Overview
This project detects car damage from images using a YOLOv8 object detection model.

## ⚙️ Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Gradio
- Google Colab

## 📊 Model Performance
- Precision: ~0.35
- Recall: ~0.34 
- mAP50: ~0.28

## 🚀 Features
- Upload image and detect damage
- Bounding box visualization
- Deployed web app

## 🧠 Learnings
- Data annotation and preprocessing
- Model training and evaluation
- Handling false positives using negative samples
- Deploying ML models using Gradio
